安装步骤：
1、将pe.sql用phpmyadmin导入数据库
2、在lib/config.inc.php文件中设置数据库参数，注意都要保存为utf8无bom形式
3、设置data目录、files/attach目录为可读写（777权限）
4、以上3步完成后即可完成安装。

前台地址：
域名/index.php
后台地址：
域名/index.php?core-master

默认管理员：
用户名：peadmin
密码：peadmin
